# Portfolio Website

## About

This is my personal portfolio website built to showcase my skills, projects, and achievements as a Front-End Developer.

## Features

* Responsive design
* Modern user interface
* Project showcase section
* Skills section
* Contact form
* Mobile-friendly layout

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap

## Installation

1. Clone the repository.
2. Open the project folder.
3. Run `index.html` in your browser.

## Project Structure

* `index.html` – Main webpage
* `style.css` – Styling
* `script.js` – Functionality
* `assets/` – Images and other resources
